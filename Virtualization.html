Virtualization Services

Lantrasoft brings expert Virtualization services to our Clients by helping to virtualize their own environments. Our Virtualization architects possess ample competency and experience in building, deploying, running and maintaining virtual environments. We help our Clients in simplifying their IT Infrastructure with Virtualization as a first step and eventually leading to an optimized Infrastructure in terms of size, computing capacity, performance aspects and so on

By virtualizing, we prepare our Customers to migrate into the Cloud Platforms (should the need arise).

Virtualization Services

The following are Service Desk's Key Performance Areas (KPA) that could be tailored and customized to suit the customer's need.

  • Virtual Infrastructure
    • Environment Provisioning on the Virtual Infrastructure - VMware Servers on ESXi Hosts
    • VMware Tools Installation
    • Segmentation into Lab and Production VC Data Centers
    • Environment Health Checks
    • Environment Consolidation - Cost and Resource Optimization of the underlying ESXi Hosts
  • Virtual Desktop
    • Upgrade the VDA agent to the latest release LTSR
    • Update the golden image and install core components
    • Create new sites for the Production and Disaster Recovery environments
    • Configure beacon for internal and external traffic
    • Create machine catalog and deployment groups
    • Manage and troubleshoot the VDI Infrastructure
  • Virtual Apps
    • Install and upgrade the XenApp Server Farm to the latest version LTSR
    • Install and publish application and desktop
    • Manage and validate DB cluster and DB instances
    • Work with internal and external business parties for hosting business critical applications
    • Create and manage polices for the Citrix Server
    • Access Control, Proactive Monitoring, Project Delivery and Operational tasks
  • Citrix Environment Provisioning
    • Upgrade PVS Console and Server to the latest version LTSR
    • Install and configure target imaging software
    • Create different environments and vDisks pertaining to the environments such as Preprod, QA and Production
    • Install updates and application on vDisk version
    • Provision Servers on-demand and deploy to Collection
    • Host and manage vDisk in the Disaster Recovery site
    • Maintain Templates and Servers
  • Maintenance Tasks
    • Daily health checks for PVS Severs and XenApp Farm
    • Address alerts and taking necessary action to resolve the alert condition
    • vDisk Update, Merge and Backup
    • Monitor network bandwidth and check latencies for ICA sessions
    • Troubleshoot XenApp and VDI user issues
    • Root Cause Analyses - Identifying and routing Network Latency, DB Utilization, User Processes consuming system resources, etc...
